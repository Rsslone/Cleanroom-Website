# 迁移整合包

## 概述

Cleanroom Loader 与绝大多数 Forge 模组兼容，通常可以直接运行而无需额外操作。<br>
然而，部分模组仍属于**已弃用、冗余或完全不兼容**的范畴——这要么是因为它们已被更好的替代品取代，要么是因为它们会引发严重问题。<br>
为了让整合包适配 Cleanroom Loader，请查阅以下列表并在必要时进行替换。<br>

## 性能/内存优化模组
对于 **Minecraft 1.12.2** 最重要的优化模组总结如下：
- **[CensoredASM](https://www.curseforge.com/minecraft/mc-mods/lolasm)**：针对 Minecraft 1.12.2 的巨大优化。
- **[VintageFix](https://www.curseforge.com/minecraft/mc-mods/vintagefix)**：改善 Minecraft 1.12.2 的加载时间和内存使用。
- **[Universal Tweaks](https://www.curseforge.com/minecraft/mc-mods/universal-tweaks)**：一个社区项目，旨在将各种错误修复和调整整合为 Minecraft 1.12.2 的统一解决方案。
- **[Stellar Core](https://www.curseforge.com/minecraft/mc-mods/stellarcore)**：提供广泛的错误修复、性能改进以及跨多种模组的附加功能。
- **[Gnetum](https://www.curseforge.com/minecraft/mc-mods/gnetum)**：通过将 HUD 更新分散到多帧来提升性能。
- **[Fixeroo](https://www.curseforge.com/minecraft/mc-mods/xp-orb-clump)**：包含了错误修复和经验球优化。
::: info 注意 {id="note"}
Universal Tweaks 取代了许多模组，因此这里不再赘述。您可以通过游戏内警告或查看 [**不兼容模组列表**](https://legacy.curseforge.com/minecraft/mc-mods/universal-tweaks/files/6953147) 来确认这一点。
:::

## 渲染优化模组
对于 **Minecraft 1.12.2** 最突出的渲染优化整合包括：

- **[Optifine](https://optifine.net/adloadx?f=preview_OptiFine_1.12.2_HD_U_G6_pre1.jar&x=c2f6)**
- **[Celeritas](https://github.com/kappa-maintainer/Celeritas-auto-build/releases)** (需要 Cleanroom Loader，并且在与 CensoredASM 同时使用时需要确保使用 CensoredASM 5.30+ 版本)
- **[Nothirium](https://www.curseforge.com/minecraft/mc-mods/nothirium)** (同时需要 **[Naughthirium](https://www.curseforge.com/minecraft/mc-mods/naughthirium)**)
::: info 注意 {id="note"}
**_Celeritas_** 和 **_Nothirium_** 会降低内存占用并改进整体内存负担，而 **_Optifine_** 可能会增加内存占用并延长加载时间。
:::

### 压力测试结果

在模拟的高负载基地场景中，我们对上述模组进行了基准测试，并根据使用的 GPU 对其性能进行了排名：<br>

- **NVIDIA GPU** → `Nothirium > Optifine > Celeritas`
- **AMD GPU** → `Celeritas > Optifine > Nothirium`
- **Intel 集成显卡** → `Celeritas | Optifine` <br> <br>
::: info 注意 {id="note"}
性能因硬件和驱动程序而异。请测试所有三个模组，以确定哪个能在操作系统上提供最佳效果。<br>
:::
